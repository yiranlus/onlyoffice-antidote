# onlyoffice-antidote

Antidote plugin for [ONLYOFFICE](https://www.onlyoffice.com/).

## Build Instructions

1. Clone the repository
   ```
   git clone https://github.com/onlyoffice/onlyoffice-antidote.git
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Build the plugin
   ```
   npm run build:plugin
   ```
4. Install the plugin manually to onlyoffice 

Note: The plugin has only be tested with Desktop version of ONLYOFFICE.
